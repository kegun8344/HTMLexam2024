Koding på Samme Dokument med GitHub

Github øker effektiviteten når en gruppe skal kode på samme dokument. Noe viktig å påpeke er at Github gir mulighet for at brukere kan jobbe samtidig uten å overskrive andres koder. Effektiviteten og selvstendigheten av å kode samtidig kommer av "branches", som er egne kopier av hoved filen, der brukere kan jobbe selv uten å påvirke hovedbranch, ofte kalt "main" branch.

Sporing av koder

Alle branches som blir lagd, kan lastes inn på hoved filen via "commits". Der kan alle se hvem som har gjort hva lagt inn hva av koden, og det som er endret.
